Joysticks may be used for players 1 and 2. The keyboard is also an option:

Player 1:
arrows - movement
A and S: buttons A and B
1: toggle hitbox view

Player 2:
I,J,K,L - movement
T and Y: A and B

A lot is hard coded right now, like player selections. This is temporary.

To quit, Ctrl+C on the console box.  It'll be cleaned up eventually.

Nothing should be required from the user to run it besides a Windows XP or newer box.
It can be compiled for linux, but a makefile would need to be made. there is no reason
it would not run, however.

This was developed using the Allegro game library and was coded from scratch in C++.
